# Dotfiles
Applied using stow.
Arch linux was used, might not work well under others linux distributions that are not rolling release.

There are configs and scripts for the following softwares
- zsh
- bspwm
- rofi 
- stow
- polybar
- mpd 
- alacritty
- flameshot
- dunst
- nvim
- ranger
- picom
- nitrogen
